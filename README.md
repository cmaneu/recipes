## Le site web

Le site est généré à partir des fichiers Markdown des recettes (aucun contenu n'est dupliqué) et déployé automatiquement sur GitHub Pages à chaque `push` sur `main` via le workflow [`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml).

```bash
npm install      # installer les dépendances
npm run dev      # serveur de développement (http://localhost:4321/recipes)
npm run build    # générer le site statique dans dist/
npm run preview  # prévisualiser le build
```

> Pour activer le déploiement, réglez **Settings → Pages → Build and deployment → Source** sur **GitHub Actions**.
